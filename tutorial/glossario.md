# Glossário - Termos Técnicos

## A

**API (Application Programming Interface)**
Interface que permite comunicação entre diferentes sistemas de software.

**Autoprefixer**
Ferramenta que adiciona prefixos de navegadores ao CSS automaticamente.

## B

**Build**
Processo de compilar e otimizar código para produção.

**Bundle**
Arquivo final que contém todo o código JavaScript/CSS da aplicação.

## C

**Component (Componente)**
Bloco reutilizável de UI com template, lógica e estilos.

**Composition API**
API moderna do Vue 3 usando `<script setup>`.

**CSS Scoped**
CSS que aplica apenas ao componente atual, não globalmente.

## D

**Dependencies (Dependências)**
Bibliotecas necessárias para o projeto funcionar.

**DevDependencies**
Ferramentas necessárias apenas durante desenvolvimento.

**DOM (Document Object Model)**
Representação em árvore do HTML no navegador.

## E

**ESM (ES Modules)**
Sistema moderno de módulos JavaScript (`import/export`).

## F

**Framework**
Estrutura que fornece fundação para construir aplicações.

## H

**HMR (Hot Module Replacement)**
Atualização instantânea sem reload completo da página.

**Hook**
Função especial executada em momentos específicos do ciclo de vida.

## I

**Import**
Trazer código de outro arquivo para uso.

**Interpolation (Interpolação)**
Inserir valores dinâmicos no template: `{{ valor }}`.

## N

**Navigation Guard**
Função executada antes/depois de navegar entre rotas.

**npm (Node Package Manager)**
Gerenciador de pacotes JavaScript.

## P

**Package.json**
Arquivo que define configurações e dependências do projeto.

**PostCSS**
Ferramenta para processar CSS com plugins.

**Props (Properties/Propriedades)**
Dados passados de componente pai para filho.

## Q

**Query Parameters**
Dados na URL após `?`: `/contato?origem=banner`

## R

**Reactive (Reativo)**
Dados que quando mudam, atualizam a UI automaticamente.

**ref()**
Função que cria variável reativa no Vue 3.

**Router**
Sistema de navegação entre páginas em SPAs.

**Route (Rota)**
Caminho URL e componente associado.

## S

**SPA (Single Page Application)**
Aplicação que carrega uma vez e atualiza conteúdo dinamicamente.

**Scoped Styles**
Estilos CSS aplicados apenas ao componente atual.

## T

**Tailwind CSS**
Framework CSS baseado em classes utilitárias.

**Template**
Seção `<template>` que contém o HTML do componente.

**Tree-shaking**
Remoção de código não utilizado no bundle final.

## U

**Utility-first CSS**
Abordagem de usar classes pequenas e específicas (Tailwind).

## V

**Vite**
Build tool moderna e extremamente rápida.

**Vue.js**
Framework JavaScript progressivo para UIs.

**Vue Router**
Biblioteca oficial de roteamento para Vue.js.

**v-if**
Diretiva que renderiza elemento condicionalmente.

**v-for**
Diretiva que renderiza lista de elementos.

**v-bind (:)**
Vincula atributo a valor dinâmico.

**v-on (@)**
Escuta eventos DOM.

## W

**Watcher**
Observador que executa código quando dados mudam.
